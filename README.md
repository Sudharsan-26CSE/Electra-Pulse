# Electra Pulse 🗳️

**Electra Pulse** is a comprehensive, modern web application designed to be your ultimate Election Hub. It helps users track their voter readiness, locate nearby polling stations, stay informed with candidate information, and navigate the election process seamlessly.

## 🌟 Features

- **Interactive UI**: A fully responsive, modern design with smooth animations using Framer Motion style transitions.
- **Dark Mode & High Contrast**: Built-in accessibility features with support for Light, Dark, and High Contrast modes.
- **Predictive Dashboard**: A personalized home screen showing your voter readiness and important updates.
- **Polling Station Locator**: Find and view details about your designated polling station.
- **Candidate Profiles**: Learn about the candidates running in your constituency.
- **Voter Toolkit**: Resources and guidelines to help you prepare for election day.
- **Secure Authentication**: Clean login and registration flows.

## 🛠️ Technologies Used

- **HTML5**: Semantic structure for accessibility and SEO.
- **CSS3**: Custom properties (variables), Flexbox, and custom animations.
- **JavaScript**: Handles UI interactivity, theme switching, and state management.
- **Lucide Icons**: Beautiful and consistent iconography.
- **Google Fonts**: Uses the 'Inter' typeface for a clean, modern aesthetic.

## 🚀 Getting Started

To run the application locally, simply clone the repository and open `index.html` in your web browser.

```bash
git clone https://github.com/Sudharsan-26CSE/Electra-Pulse.git
cd Electra-Pulse
# Open index.html in your preferred browser
```

## 📱 Responsiveness

The application is designed with a "mobile-first" approach, optimizing the layout for mobile devices, making it feel like a native mobile application while remaining fully functional on desktop environments.
